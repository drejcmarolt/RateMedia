Z razvojem digitalnih medijev in hitri rasti platform za pretočno predvajanje vsebine, se uporabniki vsakodnevno srečajo z ogromnimi števili filmov in serij. Kljub temu pa še zmeraj ni enotnega spletnega mesta, kjer si lahko uporabniki sledijo vsebine, ki so jih že pogledali, jih ocenili, komentirali in delili mnenja z drugimi. 

Cilji 

Cilj našega projekta RateMedia je ustvariti informacijski sistem, ki bo uporabnikom omogočal: 

- Vodenje osebne zbirke filmov, serij in drugih medijev 

- Zagotoviti pregledno bazo filmov in drugih medijev 

- Komentiranje in ocenjevanje vsebine 

- Sledenje drugim uporabnikom 

- Odkrivanje vsebin na podlagi ocen in priporočil drugih uporabnikov/skupin
